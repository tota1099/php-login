PHP Login Project with clean architecture and tests

## Dependencies

- PHP version 8.0.0

## Install dependencies

```bash
$ php composer.phar install
```

## Run tests

```bash
$ ./vendor/bin/phinx migrate -e testing
$ ./vendor/bin/phpunit --testdox
```
